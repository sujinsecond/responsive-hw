::before 가상 요소란?

CSS의 ::before 가상 요소는 선택된 요소의 내용 앞에 콘텐츠를 추가합니다. 이것은 실제 HTML에 추가되지 않고 CSS를 통해 시각적으로만 표시됩니다.

기본 사용법

선택자::before {
    content: "추가할 내용";
    /* 다른 CSS 속성들 */
}

예제

아이콘 버튼

.icon-button::before {
    content: "★";
    margin-right: 8px;
}

화살표 버튼

.arrow-button::before {
    content: "→";
    margin-right: 8px;
}

호버 언더라인 효과

여기에 마우스를 올려보세요 
.underline-text::before {
    content: "";
    position: absolute;
    bottom: -2px;
    left: 0;
    width: 0;
    height: 2px;
    background-color: #ff6b6b;
    transition: width 0.3s;
}

.underline-text:hover::before {
    width: 100%;
}

장식적 요소

섹션 제목

.decorated-heading::before {
    content: "";
    position: absolute;
    left: 0;
    top: 0;
    height: 100%;
    width: 5px;
    background-color: #6bc46b;
    border-radius: 2px;
}

SCSS에서의 &::before 사용법

SCSS에서는 중첩 선택자를 사용할 수 있으며, & 기호는 부모 선택자를 참조합니다: 

.button {
    background-color: #4e7bff;
    
    &::before {
        content: "★";
        margin-right: 8px;
    }
    
    &:hover {
        background-color: #3a5cb8;
    }
}

이 SCSS 코드는 다음 CSS로 컴파일됩니다:

.button {
    background-color: #4e7bff;
}

.button::before {
    content: "★";
    margin-right: 8px;
}

.button:hover {
    background-color: #3a5cb8;
}
